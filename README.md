# Neurosil UGV Project Website

This repository contains a responsive and interactive website developed for **Neurosil** to present the **UGV (Unmanned Ground Vehicle)** project. The website provides a professional platform to showcase the project's vision, features, services and products.

## Features

* Responsive and mobile-friendly design
* Modern and professional user interface
* Project overview and objectives
* UGV features and capabilities
* Smooth navigation and interactive sections
* Clean and organized layout

## Technologies Used

* HTML
* CSS
* JavaScript

This website was created as part of the **Neurosil UGV Project** to effectively demonstrate the capabilities and innovations of the Unmanned Ground Vehicle through an engaging and user-friendly web experience.
